# Cursor Glow ✨
Beautiful cursor glow + particle trail effect for modern websites.  
![demo](./demo/demo.gif)

🌎 Language

- [English](./README.md)
- [한국어](./README.ko.md)



## Install
```bash
npm install cursor-glow
```

## Usage
```js
import cursorGlow from "cursor-glow"

cursorGlow()
```


## Options
```css
cursorGlow({
  colorA:"#00ffff",
  colorB:"#8b5cf6",
  glowSize:120,
  trailCount:16
})
```


## React
```jsx
import CursorGlow from "cursor-glow/react/CursorGlow"

function App() {
  return (
    <>
      <CursorGlow />
      <h1>Hello</h1>
    </>
  )
}
```


## Features
glow cursor  
particle trail  
hover magnet effect  
click pulse  
lightweight  
zero dependencies

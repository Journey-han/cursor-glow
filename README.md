# Cursor Glow ✨
Beautiful cursor glow + particle trail effect for modern websites.


## Install
    npm install cursor-glow


## Usage
```js
import cursorGlow from "cursor-glow"

cursorGlow()
```


## Options
```css
cursorGlow({
  colorA:"#00ffff",
  colorB:"#8b5cf6",
  glowSize:120,
  trailCount:16
})
```


## React
```jsx
import CursorGlow from "cursor-glow/react/CursorGlow"

<CursorGlow />
```


## Features

glow cursor  
particle trail  
hover magnet effect  
click pulse  
lightweight  
zero dependencies
